macro_cavity¶

cavity_characterisation¶

MODULE: Contains functions used to characterize the cavity response for the macro-cavity experiment.

Main funtion: cavity_characterisation does the whole characterisation, see documentation or example notebook

relevant auxiliary functions:

cav_trans: cavity transmission function

fit_cavity_transmission: fits data to cavity transmission

levitopy.data_analysis.macro_cavity.cavity_characterisation.cavity_characterisation(date, filename_bg, filename_signal, channel_refl=1, channel_scan=4, channel_tran=2, directory='default', gain_tran=50, input_power=5e-05, time_step=8e-08, save_figures=False)¶
Does the whole enchilada:

1 Imports the data and converts appropriate axes 2 Fits the transmission to the cavity response function 3 plots the fit (saves it if ‘save_figures’=True’ 4 Calculates the coupling efficiency from the reflected light

Inputs:

channel_refl int channel_scan int channel_tran int date str directory str where is the data * filename_bg str filename_signal str gain_tran int in db input_power float in Watt time_step float in s save figures bool

  • if the directory is set to its default the directory is ‘//nas01/PNO/MEMBERS/Nadine/

Labdata/RawData_Oscilloscopes/’+date+’/’

Returns

a dictionary

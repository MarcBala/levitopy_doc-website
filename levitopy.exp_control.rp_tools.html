class levitopy.exp_control.rp_tools.red_pitaya.RedPitayaPLL2(n_pll=1, **kwargs)¶

Bases: levitopy.exp_control.rp_tools.red_pitaya.RedPitayaPLL

Modified version by Jan Gieseler, based on the modification by Gerard Conangla

allows to read the actual frequency of the PLL

(changed register for freq to a single one instead of spitting and putting back together)

pll_JG_v1.bit

  • read out instantaneous freq for both PLLs

pll_JG_v2.bit

  • read out the low pass quadrature X1 and Y1 for PLL1 from register (Y1 is usually zero when the PLL is locked!)

  • read out the phase for PLL1 from register, i.e. the error signal from the PLL

pll_JG_v3.bit

  • read out of phase now returns only 14 bits, since the first two are redundant
